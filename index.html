NoInternetConnection.exe(ppt)

This presentation is a very cool website. We have two versions. NoInternetConnection mk1, and NoInternetConnection mk2. This supports for all presentation programs.

NoInternetConnection mk1

NIC mk1(short form of NoInternetConnection) is a very nice offline slide mimicking the functions of an email client. It is absolutely free to download and easy to use. If possible, i will even make the same NIC email accessible to internet(coming soon). And also, these slides don't look like much until you start a slideshow of it. This is made using PowerPoint 2019.

NoInternetConnection mk2

NIC mk2 once again is a very nice offline slide mimicking the functions of an email client. It has some new features and an improvised theme compared to the mk1 version. The newly available features are, the logon screen(The textbox, Sign In and premium account doesn't work because of the offline functionalities) and a chatting features called "Mail Quarrels" This is made using PowerPoint 2019.
